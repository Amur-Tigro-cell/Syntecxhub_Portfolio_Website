# Subro Sutradhar - Frontend Developer Portfolio

Welcome to my cyberpunk-themed portfolio website! This project showcases my skills and projects as a frontend developer with a futuristic, neon-lit aesthetic.

## 🚀 Live Demo

[Portfolio Website Link](https://amur-tigro-cell.github.io/Syntecxhub_Portfolio_Website/)

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Modern styling with animations and effects
- **JavaScript** - Interactive elements and functionality
- **Google Fonts (Orbitron)** - Futuristic typography

## ✨ Features

- **Cyberpunk Design**: Neon-lit interface with terminal-style elements
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Glitch effects, hover states, and transitions
- **Modern Sections**: About, Skills, Projects, and Contact
- **Terminal Interface**: Unique command-line inspired contact section

## 🎨 Design Elements

- **Color Scheme**: Dark backgrounds with neon cyan, magenta, yellow, and green accents
- **Typography**: Orbitron font for that futuristic feel
- **Effects**: Glitch animations, glowing borders, matrix-style backgrounds
- **Layout**: Clean grid-based responsive design

## 🚀 Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Amur-Tigro-cell/Syntecxhub_Portfolio_Website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Syntecxhub_Portfolio_Website
   ```

3. Open `index.html` in your web browser or use a live server extension

## 📱 Sections Overview

- **Hero**: Eye-catching introduction with glitch effects
- **About**: Personal information and statistics
- **Skills**: Technical proficiency display with progress bars
- **Projects**: Showcase of featured projects

To personalize this portfolio for your own use:

3. Adjust color scheme in CSS variables
